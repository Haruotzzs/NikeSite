# NikeSite Fullstack Project

## Installation & Start

This project has **frontend** and **backend** parts. To run it, you need **Node.js** and **npm** installed.

### Install dependencies

Navigate to each folder (`frontend` and `backend`) and install dependencies:

```powershell
cd FULLSTACK/backend
npm install

cd ../frontend
npm install
```

### Start the project

From each folder, run:

```powershell
npm run start
```

> Use **PowerShell** or any terminal to run the commands.

## Frameworks and Libraries

All used frameworks and libraries are listed in the `package.json` files:

* **Backend**: check `FULLSTACK/backend/package.json`
* **Frontend**: check `FULLSTACK/frontend/package.json`

## Notes

* Make sure your backend and frontend are running in separate terminals.
* This project requires Node.js version ≥ **14.x** (or your compatible version).
* Do **not** push `node_modules` to GitHub — they are ignored via `.gitignore`.
